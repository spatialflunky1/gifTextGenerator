# Gif Text Generator
Adds custom meme like impact font text to a gif


## Features:
- [x] Add centered impact text
- [x] Resize
- [x] Top and bottom text
- [x] Tkinter GUI
- [ ] Working progress bar
- [ ] Display Gif in GUI
- [ ] Custom font
- [ ] Auto sized font
- [ ] Multiple templates



## How to Run
In terminal run:
```
python3 main.py
```

## Dependencies:
* Python 3

* Tkinter (Python module)

* Pillow (Python module)

* ttkthemes (Python module)
